Status of Implementation¶

Here is a list of the status of implementation for each view.

encode is considered the action of taking a gif, jpeg, tiff or otherwise non-DICOM file, and encoding it into DICOM.

decode is considered reading a DICOM file and outputting the view code for that particular image.
