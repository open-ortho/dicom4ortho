Feature Roadmap¶

See the open issues for a list of proposed features (and known issues).

v1.0¶

  • Lightweight minimalistic enpoint to lad metadata and images from FHIR Bundle.

  • Full send DICOM images to PACS.

v0.4¶

  • Import metadata from FHIR Bundle as file format.

v0.3¶

  • Minimal DICOM send to PACS implementation

v0¶

  • Load single PNG/JPG file.

  • Manually set metadata from filename, or hard code.
